# College Lab: Switches Configuration – VLANs

## 🧭 Overview
This lab focuses on configuring VLANs and understanding their role in network segmentation and management.  
It covers the creation of different VLAN types, assigning interfaces, configuring management VLANs, and enabling trunk links between switches.

---

## ⚙️ Tools Used
- Cisco Packet Tracer  
- Cisco IOS CLI  

---

## 🧩 Key Topics
- Default VLANs  
- Data and Management VLANs  
- Native VLANs and Trunking  
- VLAN creation and deletion commands  

---

## 🧠 Skills Practiced
- Creating and naming VLANs  
- Assigning switch ports to VLANs  
- Configuring trunk ports between switches  
- Managing VLANs using CLI and GUI (Config tab)  

---

## 🚀 Outcome
Successfully configured and tested VLAN operations, management access, and trunking between multiple switches.  
Learned how VLAN segmentation isolates traffic and improves security and management in enterprise networks.
